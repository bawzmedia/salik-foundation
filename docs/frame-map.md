## TRANSITION RULES

Every scene's **last 15–20 frames** must visually match the **first 15–20 frames** of the next scene. Here's how each bridge works:

### Prologue Bridges
- **P1 → P2:** End on harsh desert light hitting idols at angle. P2 starts with same harsh light but at night with fires.
- **P2 → P3:** End on fire dying out, darkness. P3 starts on deep darkness with faintest horizon glow.
- **P3 → R1:** End on brightening horizon. R1 starts with light exploding over that same horizon.

### Revelation Bridges
- **R1 → R2:** End on light washing over purified Kaaba. R2 starts centered on that same purified Kaaba (timelapse begins).
- **R2 → R3:** End on aerial pull-up from growing Makkah. R3 starts as that aerial becomes a stylized map with Makkah glowing.

### Map-to-Era Bridges (repeating pattern)
- **R3/Map → MAP-X:** Map is already on screen, camera zooms to new region.
- **MAP-X → Era clip:** Red dot on map dissolves/morphs into a visual element of the first era clip (campfire, sunlight, water, candlelight, etc.).
- **Era last clip → MAP-X+1:** Final frames of each era's transition clip pull up to aerial/bird's eye, which becomes the map again.

### Specific Map Dot Dissolves

|----------|----------------------|-------------------|
| MAP-1 (Scandinavia) | Campfire ember | Firelight on Viking camp |

## CLIP TYPES & GENERATION NOTES

### Standard Clips (AI Video — Kling/Runway/Sora)
P1, P2, P3, R1, R2, 1A, 1B, 2A, 2B, 3A, 3B, 4A, 4B, 5A, 5B, 6A, 6B, 6C, 7A, 7B, 7C

**21 clips** — Generate as 5-second AI video, extract 120 frames each.

### Transition/VFX Clips (Compositing or AI Video)
1C, 2C, 3C, 4C, 5C

**5 clips** — Visual morph effects. Generate as AI video or composite in After Effects. Each must end by pulling up to aerial view so it can bridge back to the map.

### Map Clips (Motion Graphics)
R3, MAP-1, MAP-2, MAP-3, MAP-4, MAP-5, MAP-6, MAP-7

**8 clips** — Animated map with golden light spreading. These should be created in After Effects, Motion, or similar motion graphics tool. Same base map asset with:
- Golden light progressively covering more territory each time
- Camera zoom to specific region
- Red pulsing dot on exact location
- Location name + century in elegant typography
- R3 is the long version (120 frames, full spread animation)
- MAP-1 through MAP-7 are short (60 frames each, zoom + dot + label)
